An AI-powered research assistant that helps users search, summarize, and analyze information using LLM APIs (like OpenRouter). Built with a simple web interface using HTML, CSS, JavaScript, and a backend service.


-  AI Smart response generation using LLM API  
- Simple and responsive frontend  
-  Fast API-based backend integration  
- Environment variable support for secure API keys  
-  Easy deployment-ready structure  

---



- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js / Python (Flask/FastAPI if used)  
- AI API: OpenRouter / LLM APIs  
- Version Control: Git & GitHub
